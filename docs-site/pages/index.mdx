# KubeStellar Console Integrations

Enhance your multi-cluster Kubernetes management with production-ready integrations for the **KubeStellar Console** — an AI-powered, adaptive dashboard.

**LFX Mentorship Project**: CNCF - KubeStellar: Integration and ecosystem development specialist (2026 Term 1)

## What is KubeStellar Console?

KubeStellar Console (kc) is a proactive, AI-powered multi-cluster Kubernetes dashboard that:

- **Adapts to how you work** using Claude AI to observe usage patterns
- **Auto-swaps dashboard cards** based on your current focus
- **Streams real-time events** via WebSocket from multiple clusters
- **Uses MCP servers** (kubestellar-ops, kubestellar-deploy) to fetch cluster data

[Learn more about Console →](https://github.com/kubestellar/console)

## Quick Start

```bash
# 1. Install KubeStellar Console
git clone https://github.com/kubestellar/console.git
cd console
./scripts/prod.sh

# 2. Clone kss-demo (integrations)
git clone https://github.com/kubestellar/kss-demo.git
cd kss-demo

# 3. Install integrations (merge into your console fork)
./scripts/install-console-integrations.sh

# 4. Configure integrations in Console .env
# PROMETHEUS_URL=http://localhost:9090
# GRAFANA_URL=http://localhost:3000
# GITHUB_TOKEN=ghp_xxx
```

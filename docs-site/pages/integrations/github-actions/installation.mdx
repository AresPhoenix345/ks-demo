# GitHub Actions — Installation

## Prerequisites

- GitHub Personal Access Token (PAT) with `actions:read` and `actions:write` scope
- For workflow_dispatch: `workflow` scope

## Create GitHub Token

1. GitHub → Settings → Developer settings → Personal access tokens
2. Generate new token (classic)
3. Scopes: `repo`, `workflow`

## Configure in Console

Add to `.env`:

```bash
GITHUB_TOKEN=ghp_xxxxxxxxxxxxxxxxxxxx
```

## Merge Integration Code

```bash
cp -r kss-demo/console-integrations/pkg/integrations/github console/pkg/integrations/
cp kss-demo/web-cards/GitHubWorkflowsCard.tsx console/web/src/components/cards/
```

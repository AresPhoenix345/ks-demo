# Prometheus + Grafana Integration

Add historical metrics, alerts, and embedded Grafana dashboards to KubeStellar Console.

## Features

- **Prometheus Metrics Card** — Query PromQL and display metrics (CPU, memory, latency)
- **Grafana Dashboard Card** — Embed Grafana dashboards in Console
- **Alert Rules Card** — Show active Prometheus alerts from AlertManager

## Architecture

```
Console Frontend (React)
        │
        ▼
Console Backend (Go)
        │
        ├──► Prometheus API (query, query_range)
        ├──► AlertManager API (alerts)
        └──► Grafana API (dashboards, embed URLs)
```

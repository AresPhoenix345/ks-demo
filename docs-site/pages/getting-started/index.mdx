# Getting Started

This guide walks you through installing KubeStellar Console and its integrations.

## Prerequisites

- **Go 1.23+** — For Console backend
- **Node.js 20+** — For Console frontend
- **GitHub OAuth App** — For authentication
- **Docker** (optional) — For Prometheus/Grafana

## Install KubeStellar Console

1. [Console Installation](/getting-started/console-installation) — Clone, configure, run
2. Install MCP plugins: kubestellar-ops, kubestellar-deploy

## Install Integrations (kss-demo)

```bash
git clone https://github.com/kubestellar/kss-demo.git
cd kss-demo
./scripts/install-console-integrations.sh
```

This merges integration packages and cards into your Console fork.

## Run the Documentation Site Locally

```bash
cd docs-site
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000).
